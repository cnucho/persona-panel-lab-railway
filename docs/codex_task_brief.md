# Codex 작업지시서

이 저장소는 수업용 MVP입니다. 다음 단계로 발전시킬 때의 작업지시입니다.

## 목표

가상 퍼소나 좌담회 실험실을 실제 수업 운영 가능한 앱으로 개선한다.

## 현재 구현

- Express 서버
- OpenAI Responses API 호출
- 학생별 크레딧 한도
- 공동 대화장
- 퍼소나별 대화창
- 회의록 저장
- 요약본 생성
- Markdown 내보내기
- Railway 배포 설정

## 개선 작업

1. UI 개선
   - 퍼소나 카드 편집 기능
   - 회의 종류별 안내문 확장
   - 크레딧 사용량 시각화 개선

2. 데이터 안정성
   - Postgres 마이그레이션 파일 분리
   - 세션 삭제/아카이브 기능
   - 학생 코드 CSV import 기능

3. 비용 통제
   - 응답 전 예상 비용 표시
   - 남은 크레딧이 적을 때 자동으로 nano 모델 전환
   - 세션 종료 기능

4. 교육 기능
   - 제출 보고서 템플릿 생성
   - 분류 타당성 감사 체크리스트
   - 사실/추정/가치 판단 태그 표시

5. 안전 기능
   - 민감정보 입력 경고
   - 실제 전문가 대체 금지 고지
   - 특정 집단 고정관념 생성 방지 안내

## 제약

- 교사용 대시보드는 만들지 않는다.
- 학생용 단일 앱을 유지한다.
- 기본은 공동 대화장이다.
- 퍼소나별 개별 대화는 보조 기능이다.
- 비용이 커지지 않도록 라운드/응답 길이 제한을 유지한다.
