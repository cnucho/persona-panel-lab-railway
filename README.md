# 전문가 퍼소나 인터뷰 랩

전문 지식, 가치관, 이해관계, 판단 규칙을 가진 가상 전문가 퍼소나를 만들고 인터뷰하는 교육용 웹 앱입니다. 퍼소나는 실제 전문가나 실제 조사 결과를 대체하지 않으며, 특정 관점에서 어떤 쟁점과 검증 필요 항목이 드러나는지 탐색하도록 설계되어 있습니다.

## 주요 기능

- 전문가 퍼소나 카드 생성: 역할, 전문 영역, 지식, 가치/이해관계, 판단 규칙, 말하기 한계 입력
- 공동 대화장: 여러 퍼소나가 패널처럼 한 번에 발언
- 개별 대화창: 특정 퍼소나와 1:1 심층면접
- 언어 설정: 시스템 언어, 전문가 응답 언어, 보고서 언어를 별도로 선택
- 회의 유형: 전문가 좌담회, 심층면접, 델파이 라이트, 포커스 그룹, 이해관계자 테스트, 광고/서비스 반응, 분류 타당성 감사
- 비용 선택: 경제형, 균형형, 심화형
- 크레딧 제한: 사용자별 예산, 라운드 수, 출력 토큰 수, 퍼소나 수 제한
- 요약 및 파일 내보내기: Markdown, Word `.docx`, 한글 `.hwpx`
- GPT 에디터: 회의록을 제출용 보고서 초안으로 다듬어 `편집본`으로 저장
- Railway 배포와 Postgres 영구 저장 지원

## 로컬 실행

```bash
npm install
cp .env.example .env
npm start
```

브라우저에서 `http://localhost:3000`을 엽니다.

`OPENAI_API_KEY`가 없으면 모의 응답으로 실행됩니다. 모의 응답은 크레딧을 차감하지 않습니다.

## 비용 제어

기본값은 수업 운영을 기준으로 보수적으로 잡혀 있습니다.

```env
CREDIT_BUDGET_PER_USER=30000
KRW_PER_CREDIT=0.1
USD_TO_KRW=1400
MAX_PERSONAS=5
MAX_ROUNDS_PER_SESSION=8
MAX_MESSAGES_PER_SESSION=100
MAX_OUTPUT_TOKENS=700
```

앱은 요청 전에 예상 비용을 계산해 남은 크레딧보다 크면 차단하고, 응답 후 실제 사용량 기준으로 차감합니다.

- 경제형: 짧은 응답, 기본 모델, 낮은 출력 토큰
- 균형형: 수업용 기본, 기본 모델
- 심화형: 더 긴 분석, 프리미엄 모델, 비용 증가
- 요약: 별도 요약 모델 사용

모델 가격은 `src/server.js`의 `MODEL_PRICES` 추정표를 사용합니다. 실제 API 가격은 바뀔 수 있으므로 배포 전 사용하는 모델의 최신 가격에 맞게 조정하세요.

## 추천 운영 방식

- 퍼소나는 3~4명으로 시작합니다.
- 공동 대화장을 기본으로 쓰고, 꼭 필요한 경우만 개별 심층면접을 사용합니다.
- 학생에게 경제형/균형형/심화형의 비용 차이를 안내합니다.
- 회의가 길어지면 요약을 만든 뒤 새 세션으로 이어갑니다.
- 실제 조사, 실제 전문가 자문, 실제 소비자 테스트처럼 표현하지 않도록 지도합니다.

## Railway 배포

1. GitHub 저장소에 코드를 올립니다.
2. Railway에서 GitHub repo를 배포합니다.
3. Variables에 `.env.example`의 값을 설정합니다.
4. 기록을 유지하려면 Railway Postgres를 추가하고 `DATABASE_URL`을 연결합니다.
5. Railway Workspace Usage에서 별도 하드 리밋을 설정해 인프라 비용도 제한합니다.

## 교육적 주의

이 앱의 출력은 시뮬레이션입니다. 전문가 퍼소나는 주어진 카드와 대화 맥락을 바탕으로 관점을 유지하도록 설계되었지만, 실제 전문가 의견이나 실제 집단의 반응을 보장하지 않습니다. 보고서에는 "가상 퍼소나 인터뷰 결과"임을 명시하고, 중요한 주장에는 검증 필요 항목을 남겨야 합니다.
